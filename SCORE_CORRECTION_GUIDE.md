# Score Correction Guide

## Purpose
The official scorekeeper enters scores for the group, but mistakes can happen. Bear Tracker should allow corrections while preserving a clear record of what changed.

## Recommended workflow
1. Group scorekeeper enters scores during the round.
2. Players can view the live leaderboard and skins.
3. Before finalization, admin reviews missing scores and unusual scores.
4. Admin can correct any score.
5. Scorekeeper can correct their own group before finalization.
6. After finalization, corrections require admin override.

## Correction log fields
- Round ID
- Player
- Hole
- Old gross score
- New gross score
- Edited by
- Edit time
- Reason/comment

## Finalization rule
Once the admin finalizes the round, payouts, skins, money totals, and quota changes are locked. If a correction is needed afterward, the app should require an admin unlock and then recalculate results.
