# Weekly Workflow Checklist

## Before Saturday
- Confirm expected players.
- Confirm guests.
- Update handicaps if needed.
- Update quotas if needed.
- Confirm payout table.

## Saturday Morning
- Create or open round.
- Select active players.
- Assign groups.
- Assign scorekeepers.
- Test one score entry on a practice hole, then clear it.

## During Round
- Monitor missing scores.
- Remind scorekeepers to save after each hole.
- Keep paper backup for the first few live tests.

## After Round
- Confirm all 18 holes are entered for all active players.
- Review places, skins, money and quotas.
- Finalize the round.
- Export backup.
