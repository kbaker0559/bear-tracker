# Bear Tracker v0.5

Adds payout preview and quota-change preview based on Saturday place-money rules.

## Added
- Payouts/Finalize tab
- 9-15 players pay 4 places; 16+ pay 5 places
- Tie splitting for places, rounded down
- Quota increase eligibility only for players winning place money
- Quota decrease preview for players below quota

## Still prototype
- Payout amounts are editable prototype defaults; next version can import/encode the official payout spreadsheet.
- Finalize is preview-only; v0.6 will apply quota updates and start season history.
