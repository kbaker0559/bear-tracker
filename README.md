# Bear Tracker — Real React Scaffold

This is the first real React/TypeScript scaffold for Bear Tracker.

## Run locally

1. Install Node.js LTS.
2. Open a terminal in this folder.
3. Run:

```bash
npm install
npm run dev
```

## Tests

```bash
npm test
```
