# Database Connection Tests v1.9

Before using Bear Tracker during a real Saturday round:

- Open the app on your PC and phone.
- Confirm both show the same player list.
- Create a test round.
- Add two groups.
- Enter one test score from your phone.
- Confirm the PC leaderboard updates.
- Enter a correction from the PC.
- Confirm the phone reflects the correction.
- Export a backup before deleting test data.

Do not use live mode for the Saturday group until these pass.
