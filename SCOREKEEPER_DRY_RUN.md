# Scorekeeper Dry Run

Use this with one or two trusted scorekeepers before the first live Saturday.

## Test format

- 2 groups
- 3 or 4 players per group
- Holes 1-3 only
- One scorekeeper per group

## Confirm

- Scorekeeper can find the correct group.
- Gross scores are entered quickly.
- Net scores calculate correctly based on hole handicap and player handicap.
- Stableford points calculate from net score vs par.
- Quota +/- updates correctly.
- Net skins cancel when tied and award only lowest unique net score.
- Results summary can be copied or printed.
