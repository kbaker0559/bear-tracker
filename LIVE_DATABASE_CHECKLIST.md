# Bear Tracker Live Database Checklist

1. Create a free Supabase project.
2. Open SQL Editor.
3. Run `supabase-schema.sql`.
4. Copy your Project URL and anon public key.
5. Duplicate `config.example.js` as `config.js`.
6. Paste your Supabase values into `config.js`.
7. Commit and push.
8. Test from two phones/browsers.

Do not put service-role keys in this app. Use only the anon/public key.
