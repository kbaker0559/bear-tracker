# Data Backup Policy

## During Local/Prototype Mode
- Export backup after every finalized round.
- Keep a copy in OneDrive and/or another folder.
- Name backups like: `BearTracker_2026-07-11.json`.

## During Live Database Mode
- Export weekly backup after finalization.
- Export monthly full database backup.
- Keep at least one off-computer backup.

## Before Major Updates
- Export data before uploading a new app version.
- Commit and push current GitHub version before replacing files.
